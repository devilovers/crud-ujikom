# CRUD Siswa - PHP & MySQL

Project ini adalah aplikasi CRUD sederhana untuk mengelola data siswa.
Dibuat menggunakan PHP Native dan MySQL untuk keperluan Uji Kompetensi.

## ✨ Fitur
- Tambah data siswa
- Edit data siswa
- Hapus data siswa
- Tampilkan data siswa

## 🛠️ Teknologi
- PHP
- MySQL
- HTML & CSS

## ⚙️ Cara Menjalankan
1. Clone repository ini
2. Import database `siswa.sql` ke phpMyAdmin
3. Jalankan di XAMPP / localhost
